# IEC 61406 - Indentifier Generator

## SYNOPSIS

IEC_61406_Demonstrator [-xyz] <uri> <file> 

## DESCRIPTION

IEC_61406_Demonstrator will generate a QR-code aacording to IEC 61406.
    
## OPTIONS

-h , usage and options (help)
